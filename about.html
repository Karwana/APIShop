Om oss

Vi är en modern och enkel webbshop som erbjuder produkter inom mode, smycken och olika accessoarer. Här vill vi göra det lätt för dig som besökare att hitta produkter som passar dig och som ligger till ett bra pris. Sidan är byggd för att vara tydlig och enkel att använda, både på dator och mobil, så att du snabbt kan se produkter, priser och bilder.

Vårt mål

Vi har valt att fokusera på enkel design och tydlig information. Målet är att sidan ska vara lätt att navigera, att det ska vara enkelt att hitta det du söker och att produkterna ska presenteras på ett snyggt och lättförståeligt sätt. Sidan visar också hur man kan kombinera tekniska funktioner med användarvänlig design för att skapa en webbshop som fungerar i praktiken.

Vi hoppas att besökare ska tycka att sidan känns modern, enkel och rolig att använda. Genom detta projekt har vi lärt oss mycket om hur man kan bygga en webbshop som både ser bra ut och fungerar bra, och vi vill visa att även en enkel sida kan ge en bra användarupplevelse.

Info

Denna webbshop är skapad som ett skolprojekt där vi lär oss hur man kan bygga en fungerande e-handel med hjälp av HTML, CSS och JavaScript. Vi hämtar produktdata från ett externt API, vilket betyder att informationen om produkterna kan uppdateras dynamiskt utan att vi själva behöver ändra på sidan. Genom projektet har vi fått öva på hur man arbetar med externa datakällor, hur man visar produkter på ett snyggt sätt och hur man kan skapa interaktiva funktioner för användaren.

På sidan kan du se varje produkt med namn, bild och pris. Vi har också lagt till en beställningsfunktion där du kan öppna en modal för att fylla i dina uppgifter. Formulären är validerade, vilket gör att du får tydlig information om något saknas eller är fel innan beställningen skickas. På det här sättet får användaren en trygg och smidig upplevelse, samtidigt som vi får visa hur man kan hantera data och användarinmatning på ett bra sätt.
